## How to search fast

1. **Find where a symbol lives** — grep the flat table first:
   - `grep -i "PaymentForm" .claude/index/symbols.tsv`
   - by kind: `grep -P "\tmodel\t" .claude/index/symbols.tsv`
   - by area: `grep -P "\tbackend:etl\t" .claude/index/symbols.tsv`
2. **Browse a layer** — open `backend.md` or `frontend.md` (grouped by role).
3. **Only then** open the source file at the `path:line` you found.
